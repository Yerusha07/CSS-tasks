CSS Attribute Selectors

Selector Example Example description
CSS [attribute] Selector [target] Selects all elements with a target attribute
CSS [attribute="value"] Selector [target="_blank"] Selects all elements with target="_blank"
CSS [attribute~="value"] Selector [title~="flower"] Selects all elements with a title attribute that contains a space-separated list of words, one of which is "flower"
CSS [attribute|="value"] Selector [lang|="en"] Selects all elements with a lang attribute value starting with "en"
CSS [attribute^="value"] Selector a[href^="https"] Selects all <a> elements with a href attribute value starting with "https"
CSS [attribute$="value"] Selector a[href$=".pdf"] Selects all <a> elements with a href attribute value ending with ".pdf"
CSS [attribute*="value"] Selector a[href*="w3schools"] Selects all <a>elements with a href attribute value containing the substring "w3schools"
