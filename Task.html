HTML styles- CSS

CSS can be added to HTML documents in 3 ways:

  • Inline:
    • It is used to Style attribute inside HTML elements.
  • Internal:
    •  It is used to Style element in the head section.
  • External:
    •  It is by using a link element to link to an external CSS file. 
 Examples: 


      1. 
      < h1 style = "color: violet;"> This is Inline CSS < /h1> 
      2. 
        < html>
        < head>
        < style>
        body {background-color: powderblue;}
        h1   {color: blue;}
        p    {color: red;}
        < /style>
        < /head>
        < body>
        
        < h1>This is a heading< /h1>
        < p>This is a paragraph.< /p>
        
        < /body>
        < /html>
    3. 
    < html>
    < head>
      < link rel="stylesheet" href="styles.css">
    < /head>
    < body>
    
    < h1>This is a heading< /h1>
    < p>This is a paragraph.< /p>
    
    < /body>
    < /html>
